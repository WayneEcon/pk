# 动态锁定指数(DLI)构建与政策冲击效应验证

**模块版本**: 2.0.0  
**作者**: Energy Network Analysis Team  
**完成时间**: 2025-08-15

---

## 📋 模块概述

本模块是美国能源独立政策国际影响研究项目的**核心创新阶段**，旨在从"关系粘性"的全新维度深度揭示美国能源独立政策的国际影响。通过构建创新的复合指标——"动态锁定指数(Dynamic Locking Index, DLI)"，量化国家间能源贸易关系的路径依赖和转换成本，超越传统的、仅基于贸易量的分析范式。

## 🎯 分析目标

### 核心研究问题
**页岩革命是否显著改变了美国与贸易伙伴之间的能源贸易锁定格局？**

### 创新价值定位
- **理论创新**: 从"关系粘性"视角理解国际能源贸易
- **方法创新**: 构建多维度DLI复合指标，使用主成分分析确定最优权重
- **政策洞察**: 揭示基础设施专用性在政策传导中的关键作用
- **统计严谨**: 使用聚类稳健标准误，避免面板数据分析的常见陷阱

## ⚙️ 核心方法论

### 1. DLI四维度构建框架

#### 维度1：贸易持续性 (Continuity)
- **公式**: `Continuity_ijt = (从2001年到t年存在贸易的年数) / (t - 2001 + 1)`
- **含义**: 衡量关系的长期性和稳定性
- **权重**: 31.9% (PCA确定)

#### 维度2：基础设施强度 (Infrastructure) 
- **公式**: `Infrastructure_ijt = log(Trade_Volume_ijt / Distance_ij + 1)`
- **理论基础**: 贸易额/距离比值作为专用性基础设施投资的代理变量
- **含义**: 衡量高沉没成本的专用性资产导致的锁定
- **权重**: 39.4% (最高权重)

#### 维度3：贸易稳定性 (Stability)
- **公式**: `Stability_ijt = 1 / (CV_ijt + 0.1)`，其中CV为5年滚动变异系数
- **关键改进**: 消除前视偏误，只使用历史数据计算
- **含义**: 衡量关系的可靠性，波动越小锁定越强
- **权重**: 8.8%

#### 维度4：市场锁定力 (Market Locking Power)
- **进口锁定**: `HHI_供应商集中度 × 供应商份额`
- **出口锁定**: `HHI_买方集中度 × 买方份额`
- **创新改进**: 实现双向计算，既考虑进口依赖也考虑出口依赖
- **含义**: 市场集中度与份额的乘积，衡量市场结构导致的锁定效应
- **权重**: 19.9%

### 2. DLI综合指标合成

#### 权重确定方法
- **方法**: 主成分分析(PCA)
- **权重来源**: 第一主成分载荷的绝对值归一化
- **解释方差**: 34.5% (第一主成分)
- **标准化**: 四个维度均标准化为均值0、标准差1

#### 最终公式
```
DLI_ijt = 0.319×Continuity_ijt + 0.394×Infrastructure_ijt 
        + 0.088×Stability_ijt + 0.199×Market_Locking_Power_ijt
```

### 3. 准实验设计(双重差分法)

#### 实验组设定
- **处理组**: 美-加、美-墨的管道贸易关系(原油+天然气)
  - 特征: 高沉没成本的专用性基础设施锁定
  - 样本: 268个观测 (16.0%)
  
- **控制组**: 美国与沙特、卡塔尔、委内瑞拉、英国、挪威、俄罗斯、阿联酋等的海运贸易关系
  - 特征: 基础设施专用性较低，转换成本更灵活
  - 样本: 1,407个观测 (84.0%)

#### 政策冲击时点
- **时间设定**: 2011年为政策冲击年份
- **理论依据**: 页岩革命产生显著产出效应的时点

#### DID回归模型
```
DLI_ijt = α + β₁×Treatment_ij + β₂×Post_t + β₃×(Treatment_ij × Post_t) + γ×Controls_ijt + ε_ijt
```

**关键技术改进**：
- 使用**聚类稳健标准误**按国家聚类，避免面板数据的序列相关性偏误
- 这是面板数据DID分析的统计学最佳实践

## 📊 主要发现

### 核心结果摘要

#### 1. DLI综合指标的政策效应
- **DID系数**: -0.193 ⭐⭐
- **统计显著性**: p = 0.048 (5%水平显著)
- **95%置信区间**: [-0.385, -0.002]
- **实质含义**: 页岩革命显著**降低了**管道贸易关系的锁定效应

#### 2. 分维度效应分解

| 维度 | DID系数 | 标准误 | t统计量 | p值 | 显著性 | 效应方向 |
|------|---------|--------|---------|-----|--------|----------|
| **贸易持续性** | -0.065 | 0.015 | -4.46 | <0.001 | ⭐⭐⭐ | 锁定**减弱** |
| **基础设施强度** | -0.020 | 0.065 | -0.30 | 0.762 | - | 无显著效应 |
| **贸易稳定性** | +0.814 | 0.340 | +2.39 | 0.017 | ⭐⭐ | 锁定**增强** |
| **市场锁定力** | -0.027 | 0.013 | -2.11 | 0.035 | ⭐⭐ | 锁定**减弱** |

#### 3. 关键科学洞察

**🔍 发现1: 锁定效应的显著削弱**
页岩革命使管道贸易国家(加拿大、墨西哥)的DLI综合指标相对于海运贸易国家平均**降低了0.193个标准差**，表明政策确实**减弱了基于专用性基础设施的贸易锁定**。

**🔍 发现2: 能源独立的实证证据**
- **持续性下降**(-0.065)：管道贸易关系变得不再那么"铁板一块"
- **市场锁定力下降**(-0.027)：对特定供应商的依赖度降低
- 这直接印证了页岩革命增强美国能源独立性的政策目标

**🔍 发现3: 稳定性的悖论式增强**
尽管整体锁定效应下降，但贸易**稳定性反而显著提高**(+0.814)，这可能反映：
- 页岩革命带来的供应安全感使贸易更加稳定
- 美国在管道贸易中获得了更多主导权和议价能力

**🔍 发现4: 基础设施锁定的韧性**
基础设施强度维度无显著变化，表明：
- 既有的管道等专用性资产具有很强的锁定惯性
- 短期内(2011-2024)基础设施格局难以根本改变
- 政策影响主要通过市场结构和贸易模式传导

### 统计质量保证

#### 方法论严谨性
- ✅ **聚类标准误**: 按国家聚类，避免面板数据偏误
- ✅ **无前视偏误**: 稳定性计算严格使用历史数据
- ✅ **完整键值**: 数据合并包含us_role字段，避免进出口混淆
- ✅ **异常值处理**: IQR-based稳健处理，标记但不删除异常值

#### 数据规模与质量
- **面板数据规模**: 27,643个观测点
- **时间跨度**: 2001-2024年 (24年)
- **空间覆盖**: 221个贸易伙伴国(CEPII距离数据覆盖93.7%)
- **产品覆盖**: 4种能源产品
- **DID样本**: 1,675个有效观测
- **模型拟合度**: R² = 0.655 (DLI综合指标)
